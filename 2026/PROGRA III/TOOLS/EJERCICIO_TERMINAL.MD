# Ejercicio Práctico de Terminal

**Objetivo:** Poner en práctica los comandos básicos de la terminal para la navegación y manipulación de archivos y directorios.

**Instrucciones:** Abre una terminal en tu sistema operativo (PowerShell o CMD en Windows, o Bash en Linux/macOS) y sigue los pasos que se detallan a continuación.

---

## Parte 1: Navegación y Creación

1.  **Verificar tu ubicación:**
    *   Usa el comando para saber en qué directorio te encuentras actualmente.

2.  **Crear un directorio de trabajo:**
    *   Crea una nueva carpeta llamada `mi_proyecto`.

3.  **Ingresar al nuevo directorio:**
    *   Navega hacia la carpeta `mi_proyecto` que acabas de crear.

4.  **Crear estructura de archivos:**
    *   Dentro de `mi_proyecto`, crea dos carpetas: `src` y `docs`.
    *   También dentro de `mi_proyecto`, crea un archivo de texto vacío llamado `README.md`. Puedes usar el comando `touch README.md` en Linux/macOS o `echo. > README.md` en Windows.

5.  **Listar el contenido:**
    *   Muestra el contenido de la carpeta `mi_proyecto` para verificar que se hayan creado las carpetas y el archivo.

---

## Parte 2: Manipulación de Archivos

1.  **Crear y mover un archivo:**
    *   Dentro de `mi_proyecto`, crea un nuevo archivo llamado `index.js`.
    *   Mueve el archivo `index.js` a la carpeta `src`.

2.  **Copiar un archivo:**
    *   Copia el archivo `README.md` a la carpeta `docs`.

3.  **Renombrar un archivo:**
    *   Navega a la carpeta `docs`.
    *   Renombra el archivo `README.md` a `documentacion.md`.

4.  **Verificar la estructura final:**
    *   Vuelve al directorio `mi_proyecto`.
    *   Lista el contenido de `mi_proyecto` y también el contenido de las carpetas `src` y `docs` para asegurarte de que todo esté en su lugar.

---

## Parte 3: Limpieza

1.  **Eliminar un archivo:**
    *   Dentro de la carpeta `docs`, elimina el archivo `documentacion.md`.

2.  **Eliminar un directorio:**
    *   Vuelve al directorio raíz de `mi_proyecto`.
    *   Elimina la carpeta `docs`.

3.  **Eliminar todo el proyecto (¡con cuidado!):**
    *   Sube un nivel para salir de `mi_proyecto`.
    *   Elimina la carpeta `mi_proyecto` y todo su contenido.

---

¡Felicidades! Has completado el ejercicio práctico de la terminal.
